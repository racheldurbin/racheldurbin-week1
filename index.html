A friend of mine who attended Epicodus explained to me that I could work for companies whose values I admire and still be creative while working in coding and developement.

Below are the projects I created during my first week at Epicodus and a brief description of them.

  1. Hello-world is a webpage that tells you how to say "hello" in seven different languages. I created this project in the HTML language.
  2. My-first-website is a webpage listing what I hope to learn while attending Epicodus. This project was created with HTML, but includes an example of adding a README.
  3. Goodbye explains how to say goodbye in English and Spanish. Again, this project was created with HTML language only.
  4. Cookie-recipe includes the ingredients and directions on how to make the best chocolate chip cookies ever. This was the last project I only used the HTML language to create. For the first week, the goal was to understand HTML before moving on to other languages.
  5. Cupcake-shop is a webpage I created displaying three different cupcakes. I used the languages HTML and CSS in this project and included three images.
  6. Boring-lecture is a webpage I created discussing how boring lorem ipsum is to everyone in the universe. This project was also created with HTML and CSS, but only contains one image.
